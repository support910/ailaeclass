============================================================
十、已知问题与解决方案
============================================================

1. 503 错误 (A2 Hosting)
   原因: ES Module 与 CommonJS 语法冲突
   修复: A2 支持团队将 app.js 重命名为 app.cjs
   结论: 不建议在 A2 共享主机跑 Node.js，继续用 Railway

2. Unauthenticated user (上传时)
   原因: hooks.server.ts 只接受 Bearer token，前端发的是 raw token
   修复: 已修改为兼容两种格式
   文件: apps/dashboard/src/hooks.server.ts

3. Invalid key (中文文件名)
   原因: Supabase Storage key 不支持非 ASCII
   修复: sanitizeFileName 去除所有非 ASCII，保留 a-zA-Z0-9_.-
   文件: upload/download server.ts

4. Video upload 按钮禁用
   原因: UploadVideo.svelte 依赖 PUBLIC_SERVER_URL
   修复: 移除依赖，迁移到本地 API 路由
   文件: UploadVideo.svelte, presign.ts

5. Docker build 失败
   原因: corepack pnpm 版本不匹配 / @sentry/profiling-node 原生编译失败
   修复: pnpm@8.15.9 + --ignore-scripts + max-old-space-size
   文件: docker/Dockerfile.dashboard

6. Railway 域名 403
   原因: A2 Hosting Subdomains 功能强制生成 A 记录，覆盖 CNAME
   修复: 需从 cPanel Subdomains 列表删除子域名
   状态: camp.5gnumultimedia.com 正在尝试
